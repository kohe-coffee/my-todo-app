# モダンなTodoリストアプリケーション

美しいUIと豊富な機能を備えたTodoリストアプリケーションです。

## 主な機能

- タスクの追加、編集、削除
- タスクの優先度設定（高・中・低）
- カテゴリー分類
- 期限の設定
- タスクの検索機能
- 優先度、期限、カテゴリーによる並び替え
- モダンなUIデザイン

## 技術スタック

- React
- TypeScript
- Material-UI (MUI)
- date-fns

## ローカルでの実行方法

1. リポジトリをクローン
```bash
git clone [リポジトリURL]
```

2. 依存関係のインストール
```bash
cd my-todo-app
npm install
```

3. 開発サーバーの起動
```bash
npm start
```

ブラウザで `http://localhost:3000` を開くと、アプリケーションが表示されます。
